# IC-Naming JS SDK

## Installing

Using npm:

```sh
npm install @ic-naming/client
```

Using yarn:

```sh
yarn add @ic-naming/client
```

Using unpkg CDN. Access through `(window or global).IcNaming.Client`:

```html
<script src="https://unpkg.com/browse/@ic-naming/client/dist/index-umd.js"></script>
```
